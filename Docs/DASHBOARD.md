## Directory Structure

*   **_assets/**: Assets used by the game (fonts, noteskins, themes).
*   **Announcers/**: Announcer packs.
*   **BackgroundEffects/**: Lua scripts for background effects.
*   **BackgroundTransitions/**: XML/Lua for background transitions.
*   **BGAnimations/**: Background animations.
*   **Build/**: Build instructions and scripts.
*   **Characters/**: 3D character models and scripts.
*   **CMake/**: CMake configuration modules.
*   **Courses/**: Course definitions.
*   **Data/**: Data files (AI, translations, mappings).
*   **Docs/**: Documentation.
*   **extern/**: External dependencies (source code).
*   **Installer/**: Installer scripts (NSIS, etc.).
*   **NoteSkins/**: Note skins (arrows, receptors).
*   **Program/**: Compiled binaries (output directory).
*   **Scripts/**: Global Lua scripts.
*   **Songs/**: Song packs.
*   **src/**: Source code (C++).
    *   **Economy/**: Economy system implementation.
    *   **Gym/**: Gym/Training mode implementation.
    *   **Tournament/**: Tournament mode implementation.
    *   **Unified/**: Unified dashboard implementation.
*   **Themes/**: Themes (skins).
*   **Utils/**: Utility scripts.
*   **Xcode/**: Xcode project files.
