# User Guide: StepMania Network Features

Welcome to the new StepMania Network! This guide explains how to use the new Economy, Tournament, and Fitness features.

## 1. Getting Started

### Login
1.  Navigate to the **QR Login** screen from the main menu.
2.  Scan your Player Card (or type `SMX:USERID` on the keyboard).
3.  The system will authenticate you and load your **Wallet Balance**.

### The Wallet
*   Every new user starts with **50 Coins** (Welcome Bonus).
*   You earn coins by:
    *   Winning Bets.
    *   Running your machine in "Server Mode".

## 2. Competitive Play

### Betting
1.  Before playing a song, you will see the **Betting Screen**.
2.  Use Left/Right to wager coins (e.g., 10 Coins).
3.  **Win Condition:** Pass the song with **> 80% Accuracy**.
4.  **Result:**
    *   **Win:** You get your wager back + 100% profit (Double up).
    *   **Loss:** The House takes your wager.

### Tournament Ladder
*   Select **Tournament Ladder** from the main menu to see where you rank.
*   Ranks are divided into **Pro**, **Gold**, **Silver**, and **Bronze** divisions.

## 3. Fitness Mode (Gym)

*   Select **Gym Mode** from the main menu.
*   This mode offers a simplified interface.
*   Press **START** to generate a "Quick Circuit" playlist (Random songs matched to your intensity).

## 4. Industry & Governance

### Commercial Mode
*   If you are an Arcade Operator, run the game in **Commercial Mode**.
*   Every song played automatically deducts a small **Royalty Fee** from the user's wallet and sends it to the Artist's simulated wallet.

### DAO Voting
*   Access **Governance** from the Options menu.
*   Vote on proposals (e.g., "Add Song X", "Increase Fees") using your earned Voting Power.

### Server Mode
*   When not in use, leave the machine on the **Server Node** screen.
*   It will simulate processing blockchain blocks and credit your wallet with **Mining Rewards** over time.
